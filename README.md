# Now Available Page

| url                         | usage                  |
|-----------------------------|------------------------|
| /page/index                 | Show Main Page         |
| /page/building?{name}       | Show Building By Name  |
| /page/buildingGroup?{name}  | Show Building By Group |
| /page/admin/building        | Show List of Building  |
| /page/admin/building?{name} | Show Edit on Building  |
